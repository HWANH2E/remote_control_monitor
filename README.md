# remote_control_monitor

## 개발환경
- 언어/프레임워크: **C#, WPF**

## 프로젝트 상세 설명
원격 호스트 주소에 제어를 원하는 상대 IP 주소를 입력한 뒤 **설정하기** 버튼을 클릭하면 원격 제어 요청이 전송됩니다.  
요청을 받은 컴퓨터에서 **원격 제어 허용** 버튼을 클릭하면, 요청한 컴퓨터에서 마우스/키보드 등의 원격 제어가 가능해집니다.

### 구성도(흐름)
```mermaid
flowchart LR
    A[요청 PC\n원격 호스트 주소 입력] --> B[설정하기 클릭\n원격 제어 요청 전송]
    B --> C[수신 PC\n원격 제어 요청 수신]
    C --> D[원격 제어 허용 클릭]
    D --> E[요청 PC에서\n마우스/키보드 제어 가능]
```

### 기능별 구성
1. **원격 호스트 주소 입력**
   - 제어를 원하는 상대 IP 주소 입력
2. **요청 전송**
   - **설정하기** 버튼 클릭 시 원격 제어 요청 전송
3. **요청 수신 및 승인**
   - 수신 PC에서 **원격 제어 허용** 클릭
4. **원격 제어 활성화**
   - 승인 후 마우스/키보드 원격 제어 가능

### UI 구성(텍스트)
- **원격 호스트 주소 입력란**: 요청 대상 IP 입력
- **설정하기 버튼**: 원격 제어 요청 전송
- **원격 컨트롤러 주소 표시**: 요청을 보낸 PC 정보 표시
- **원격 제어 허용 버튼**: 수신 PC에서 승인
